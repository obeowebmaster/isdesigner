Elle leur permet de spécifier ensemble l’application à développer et de mieux la partager avec les architectes applicatifs, les développeurs et les Ops en charge du déploiement." img_src="images/illustrations/sirius-overview.png" img_title="Information System Designer - What is Information System Designer?" img_class="" %}
{% include block_text_image.html txt_order="order-1" title="Analyse Métier" desc="

Information System Designer assiste les analystes métiers dans le processus d’identification et de formalisation des besoins métiers :

  • Formaliser les exigences
  • Spécifier les cas d’utilisation
  • Identifier les entités métier
  • Maquetter les IHM

Une description visuelle facilite les échanges aussi bien avec les utilisateurs qu’avec les équipes de conception et de développement.
