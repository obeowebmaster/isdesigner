Le code complet de l’atelier Information System Designer est disponible sur GitHub

L’atelier Information System Designer est un ensemble packagé de plug-ins Eclipse

Information System Designer s’appuie sur Liquibase pour rétro-modéliser des bases de données existantes

Les éditeurs graphiques développés avec Sirius sont extensibles et personnalisables

Information System Designer permet de rétro-modéliser des interfaces de services définies avec OpenAPI

Les modèles créés avec Information System Designer peuvent servir pour générer du code avec Acceleo
